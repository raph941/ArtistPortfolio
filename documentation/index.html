Files Structure

- assets/popper                   (Popper Js library files used)

 

HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Main Jquery JS -->
<script src="./assets/jquery/jquery-3.5.1.slim.min.js" type="text/javascript"></script>        
<!-- Popper JS -->
<script src="./assets/popper/popper.min.js" type="text/javascript"></script> 
<!-- Bootstrap JS -->
<script src="./assets/bootstrap/js/bootstrap.min.js" type="text/javascript"></script>          
<!-- Custom JS -->
<script src="./assets/js/main.js"></script>

 

Fonts

To change the embedded font, please take a look in the head part of index.html:
  
            @import url('https://fonts.googleapis.com/css2?family=Josefin+Sans:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;1,200;1,300;1,400;1,500;1,600;1,700&display=swap');

 

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code to the top of global.css.
 
Then edit font family in *, to change the primary font
  
*{
    font-family: 'Open Sans', sans-serif;
}

Sliders

Slider Js Carousel. It an easy to implement carousel, which provides flexibility and rhobust featuers
 
Testimonial Section slider has class 'swiper-container' and each slide has '<div class="swiper-slide">' wrapper for easy find where slide html code start. If you want to remove some slides just remove div with all content like this '<div class="carousel-item">'
  
 
<div class="swiper-container" >


<div class="swiper-slide">
....
</div>


</div>

You can find more documentation here https://getbootstrap.com/docs/4.3/components/carousel
